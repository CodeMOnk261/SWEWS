# UI/UX Design Prompt — Space Weather Early Warning System (SWEWS)

Design a **research-grade web application** for an AI-powered **Space Weather Early Warning System**. The interface should look like software used in a satellite mission control center, not a traditional business dashboard. The visual identity should convey scientific precision, trust, and advanced technology while remaining clean, modern, and highly readable.

## Design Style

The aesthetic should combine inspiration from:

* NASA Mission Control
* ESA Space Weather Portal
* SpaceX Mission Control
* Apple VisionOS
* Linear.app
* Bloomberg Terminal

Keywords:

Scientific, Minimal, Futuristic, Premium, Aerospace, Clean, Professional, High-tech, Dark UI, Data-centric, Elegant.

Avoid:

* Glassmorphism overload
* Neon cyberpunk effects
* Gaming UI
* Bright gradients everywhere
* Cartoon illustrations
* Excessive shadows
* Large colorful statistic cards

The application should feel like a scientific instrument used by engineers and researchers.

# Responsive Behaviour

Desktop-first.

Large monitors should resemble mission-control software.

Tablet support is secondary.

Mobile should show simplified scientific panels without losing readability.

---

# Overall Experience

The final interface should immediately communicate:

"This is a professional aerospace research platform used for monitoring and predicting hazardous space weather events."

The design should feel premium, scientific, calm, and technically sophisticated rather than flashy or entertainment-focused. Every visualization should serve a purpose, every panel should present actionable scientific information, and the entire experience should resemble the operational software found in a real satellite mission control center.
